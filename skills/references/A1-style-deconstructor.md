# A1: Style Deconstructor

Reverse-engineer high-converting TikTok sales videos so users can recreate the style for their own products.

## Instructions

When user provides a TikTok sales script, analyze and output:

### 1. STRUCTURAL BREAKDOWN
- **Hook Type:** Problem call-out / Bold claim / POV / Curiosity gap
- **Flow Structure:** Hook → Problem → Agitation → Solution → Proof → CTA
- **Timing/Pacing:** Where are the beats? When does each section start?
- **Transitions:** How does it move from section to section?

### 2. PSYCHOLOGICAL PATTERNS
- **Core Tension:** What gap between desire and reality does it exploit?
- **Emotional Arc:** frustration → hope → urgency (or variation)
- **Identity Triggers:** Who does this speak to specifically?
- **Trust Builders:** What makes it feel authentic vs. salesy?
- **Urgency Mechanisms:** How does it create desire to act now?

### 3. LANGUAGE TECHNIQUES
- **Specific Phrases:** Exact words/phrases that create impact
- **Tone & Voice:** Casual/authoritative/vulnerable/excited
- **Rhythm Patterns:** Short sentences? Questions? Repetition?
- **Power Words:** Words carrying emotional weight
- **Authenticity Markers:** What makes it feel unscripted?

### 4. VISUAL/DELIVERY NOTES
- **What to show:** Key visual moments implied
- **Energy level:** High/medium/low
- **Camera style:** Talking head / Demo / POV / B-roll

### 5. RECREATION CONTEXT NEEDED
List what info about user's product/audience is needed to recreate this style.

### 6. REUSABLE TEMPLATE
Provide fill-in-the-blank template version of the script structure.

## Usage Note

Repeat this process for 2-3 scripts from the same creator to identify consistent patterns before proceeding to A2 (Style Synthesizer).

## Input Format

```
Please paste the TikTok sales video script/transcript you want me to analyze:

[USER PASTES SCRIPT HERE]
```
